# 校准案例与误判

## 目录

1. 消费品：产品机制与场景升级
2. 工业品：高价值场景与品类公值
3. 常见误判清单

## 1. 消费品：产品机制与场景升级

候选A强调可见产品机制，候选B强调消费场景升级时：

- 若A能直接解释“为什么更好吃/有效/可信”，有历史配称和顾客认知，优先把A作为购买理由或核心特性位置。
- 若B只是“正餐级、高端级、专业级”等等级修饰，而顾客不自然使用、同行也已普遍升级，不判为新品类。
- 可重组为：以A形成差异化，以B指导产品、环境和服务升级。

校准例：现穿羊肉串比“正餐级烤串”更接近可占据的位置；正餐化适合做经营升级方向。注意：当“现穿”已成为行业共同能力时，必须继续用产地、当天处理、明档、标准或规模证据加固。

## 2. 工业品：高价值场景与品类公值

候选A定位高价值应用场景，候选B强调安全/质量时：

- 安全通常是压力容器、医疗、工程设备等品类的准入门槛，重要但不独特。
- 高端也不能只靠形容词成立，必须翻译成高负荷、高损失、高精度、关键系统等场景，并用参数、认证、客户和运行数据证明。
- 优先结构：高价值场景定位 + 安全可靠核心价值 + 制造/检测/案例信任状。

校准例：“高端空压站配套储气罐”比“安全第一”更接近市场位置；安全应成为证明高端的核心价值。若企业没有高端项目、标准与系统配套能力，则高端定位也不成立。

## 3. 常见误判清单

1. 把技术创新当品类创新。
2. 把行业统计分类当顾客购买品类。
3. 把企业优势直接当顾客价值。
4. 把新词当新品类。
5. 把无关紧要的差异当定位。
6. 把第一家当第一品牌。
7. 把流量热度当需求趋势。
8. 把多产品和多场景当增长必需。
9. 把广告语当战略。
10. 把定位当永久静止。
11. 把安全、品质、服务、专业等公共价值当独占位置。
12. 把企业自述的百年、第一、唯一、领先当成事实。
13. 为了支持候选之一而忽略单位经济和交付能力。
14. 评分只报数字，不说明证据与不确定性。
15. 两个候选属于不同战略层级，却强行二选一。
